# LineLens

LineLens is a simple C++ command-line tool that compares two text files line by line and reports the differences between them.

The program reads both files, checks each line in order, and displays any mismatches along with their line numbers.

---

## Features

- Compare two text files line by line
- Display differing line numbers
- Show content from both files for each difference
- Detect missing lines when one file is longer than the other
- Generate a quick comparison summary
- Lightweight and easy to use
