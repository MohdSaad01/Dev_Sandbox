# 🧠 Quiz Application

A beginner-friendly C++ console application that allows users to take a multiple-choice quiz, track their score, and receive a performance evaluation at the end.

This project was built to practice fundamental C++ concepts such as classes, structures, vectors, functions, loops, input validation, and basic object-oriented programming.

---

## ✨ Features

- 📋 Simple menu-driven interface
- ❓ 10 multiple-choice questions
- 🔀 Randomized question order every time the quiz starts
- ✅ Input validation for menu and answer choices
- 🎯 Score tracking system
- 📊 Quiz statistics and percentage calculation
- 🏆 Performance evaluation
- 🔁 Replay the quiz without restarting the program
