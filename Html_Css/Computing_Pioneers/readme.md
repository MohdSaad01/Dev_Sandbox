# The Minds Behind Computing

A responsive website that highlights some of the most influential innovators in computer science and the technologies they created.

## Overview

This project explores the people and inventions that helped shape modern computing. It presents a historical timeline, detailed inventor profiles, notable quotes, and the long-term impact of their contributions.

Featured technologies include:

- UNIX
- C Programming Language
- TCP/IP
- Linux Kernel
- World Wide Web

Featured innovators include:

- Dennis Ritchie
- Ken Thompson
- Vint Cerf
- Linus Torvalds
- Tim Berners-Lee

## Features

- Responsive layout
- Modern dark-themed interface
- Historical computing timeline
- Inventor profile cards
- Hover animations and transitions
- Mobile-friendly design
- Built entirely with HTML and CSS
