# FolioOne

A clean and responsive developer portfolio website built using Bootstrap 5 and Flask. The project demonstrates responsive layouts, reusable UI components, custom styling, and server-side template rendering.

## Features

- Responsive layout
- Modern portfolio design
- Bootstrap 5 components
- Custom color palette and typography
- Hero, About, Resume, Projects, and Contact sections
- Flask-powered template rendering

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript
- Python
- Flask

## Learning Objectives

This project was created to practice:

- Bootstrap grid system
- Responsive web design
- Template customization
- Flask routing
- Static file organization
- HTML & CSS fundamentals
